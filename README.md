# 408-ai-tutor
An AI tutor system for CS 408 exam using RAG and Graph-RAG




## 1. 项目简介

**408-AI-Tutor** 是一个面向计算机 408 考研的智能教练系统，主要覆盖：

- 数据结构
- 计算机组成原理
- 操作系统
- 计算机网络

项目围绕「大模型 + 检索 + 知识图谱」构建了一套学习辅助体系，包括：

- 对教材、笔记、真题进行**检索增强生成（RAG）**，支持按章节/知识点精细检索
- 构建 408 **知识点图谱**，采用 **Graph-RAG** 策略降低幻觉和无关召回
- 设计「**费曼学习模式**」与掌握学习流程，帮助用户自我讲解、系统纠错
- 支持 **DeepSeek API**，并预留本地大模型接入接口（如 HuggingFace / llama.cpp 等）

作者对大模型辅助教育有一定兴趣，后续想将一些教育的模式引入该项目。
